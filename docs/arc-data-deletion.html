Facebook Data Deletion Instructions

ARC — Agent Remote Control

To request deletion of the data ARC collected via Facebook Login, follow these steps:

  1. Send an email to mjshuff23@gmail.com with the subject line Facebook Data Deletion Request.
  2. Include the email address or Facebook account associated with your ARC account.
  3. We will permanently delete your account, sessions, and all associated data within 30 days and confirm via email.

ARC is a self-hosted personal productivity tool. Data is stored only on your own server and is never shared with third parties.
